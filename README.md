# Secure-Software-Implementation
## 📱 Overview

This project contains two integrated Android applications:

- ⚠️ **Vulnerable App** – intentionally insecure, designed to demonstrate common Android security flaws  
- 🔒 **Secure App** – implements best practices to mitigate those vulnerabilities  

Both apps are designed to work together, allowing developers, students, and security researchers to:

- Understand real-world Android vulnerabilities  
- Practice exploitation techniques safely  
- Learn how to properly secure mobile applications
